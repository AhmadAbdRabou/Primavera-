Project Data

  • Project Data is where you upload Primavera Project in XER file format

  • Click on File Selector and choose a file, Once file loaded, the file content will be displayed in the text area

  • Next step is to convert XER file to XLSX workbook by clicking Extract Data

  • Please note that for preformance matter, not all project tables displayed, but all project tables are extracted in the XSLX file

  • Also, A particular project may or may not utilize all tables of original primavera schema tables

  • To download XLSX workbook click on Excel-File Name
